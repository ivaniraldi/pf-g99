Hito 1 - Diseño y prototipo.

1- Definicion del grupo/roles (UI/UX Designer (El que hace que todo se vea bien)-Arquitecto / Backend Planner (El cerebro del sistema)-Tech Lead / Frontend Planner (El que conecta todo))

1a.UI/UX Designer (El que hace que todo se vea sexy)

Responsabilidades:

Diseñar los bocetos de todas las vistas
Definir layout (home, login, registro, perfil, etc.)
Pensar en la experiencia del usuario (flujo lógico)
Elegir estilos (colores, tipografías, vibe del marketplace)

Herramientas sugeridas:

Figma (pro)
Draw.io
Google Slides (rápido y suficiente)

Output clave: Diseño completo de pantallas + navegación visual

1b.Arquitecto / Backend Planner (El cerebro del sistema)

Responsabilidades:

Diseñar la base de datos (tablas + relaciones)
Definir el contrato de la API REST
Pensar cómo fluye la data (usuarios, publicaciones, auth)
Decidir qué endpoints existen

Ejemplo de cosas que define:

/api/login
/api/products
/api/users/profile

 Output clave: Diagrama DB + endpoints documentados

1c.Tech Lead / Frontend Planner (El que conecta todo)

Responsabilidades:

Definir las dependencias del proyecto
Diseñar la navegación (público vs privado)
Pensar cómo manejar el estado (auth en frontend)
Alinear todo para que luego sea fácil de codear

Ejemplo:

Bootstrap
Axios
React Router Dom

Output clave: Lista de dependencias + lógica de navegación
